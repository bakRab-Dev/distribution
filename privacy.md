# プライバシーポリシー

本アプリ（以下「本サービス」といいます）は、ユーザーのプライバシー保護を重視し、以下の通り個人情報の取り扱いを規定します。

## 1. 取得する情報と利用目的
本サービスでは、以下の情報を取得し、それぞれの目的で利用します。
- **Firebase 認証情報 (UID)**: ユーザーの識別および有料プランの状態管理。
- **メールアドレス**: ログイン時の本人確認および重要なお知らせの送付。
- **Patreon 連携データ**: 支援状態の確認のためにPatreon APIを利用します。当方がクレジットカード番号等の機密情報を直接取得・保持することはありません。
- **キャッシュデータ**: 検索の利便性向上のため、取得済みデータをユーザーの端末内にのみ保存します。

## 2. 外部サービスの利用
- Firebase (Google LLC)
- Patreon, Inc.
- GitHub

## 3. データの管理義務（重要）
本サービスは外部サイトの情報を取得し、端末内に一時保存（キャッシュ）します。
- 当方のサーバーに取得データを永続的に保存（ミラーリング）することはありません。
- **参照先サイトにおいて削除されたデータについては、本アプリ内においてもユーザー自身の操作により速やかに削除し、保持し続けないようにしてください。**
- データの管理権限および削除義務はすべてユーザーに帰属します。
